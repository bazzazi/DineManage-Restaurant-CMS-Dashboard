# 🍽️ Restaurant CMS

سیستم مدیریت محتوای سبک و واکنش‌گرا برای وب‌سایت رستوران، توسعه‌یافته با HTML، CSS و JavaScript خالص.

---

## 📌 معرفی

این پروژه یک پنل ساده برای مدیریت محتوای وب‌سایت رستوران است. هدف، ارائه یک رابط سبک، سریع و قابل توسعه بدون استفاده از فریم‌ورک‌های سنگین است.

---

## ✨ امکانات

- طراحی واکنش‌گرا (Responsive)
- رابط کاربری ساده و کاربردی
- بدون وابستگی به فریم‌ورک‌ها
- ساختار سبک و قابل توسعه
- مناسب برای پروژه‌های آموزشی و واقعی

---

## 🛠️ تکنولوژی‌ها

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 📁 ساختار پروژه

```

cms-build/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── app.js
└── assets/

````

---

## 🚀 اجرا

```bash
git clone https://github.com/your-username/restaurant-cms.git
cd restaurant-cms/cms-build
````

سپس فایل `index.html` را در مرورگر باز کنید.
