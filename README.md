# 🍽️ رستوران‌یار — پنل مدیریت محتوای حرفه‌ای رستوران

> یک سیستم مدیریت محتوای (CMS) اختصاصی و کاملاً شخصی‌سازی‌شده برای صنعت **رستوران، کافه و فست‌فود** — با طراحی مدرن، تیره و حرفه‌ای، الهام‌گرفته از بهترین داشبوردهای جهانی.

**طراحی و توسعه توسط: محمدعلی بزازی** ✨

---

## 📖 درباره پروژه

**رستوران‌یار** یک پنل مدیریت محتوای کاملاً اختصاصی است که با تمرکز بر نیازهای واقعی صنعت رستوران‌داری طراحی شده است. این پروژه با استفاده از **HTML, CSS و JavaScript خالص** (بدون هیچ فریمورک یا کتابخانه خارجی) ساخته شده تا سبک، سریع و قابل سفارشی‌سازی باشد.

این پنل به مدیران رستوران امکان می‌دهد تا تمام جنبه‌های کسب‌وکار خود را در یک رابط کاربری زیبا و کارآمد مدیریت کنند: از سفارش‌های لحظه‌ای گرفته تا تحلیل فروش، مدیریت منو، رزرو میز و کارکنان.

---

## ✨ ویژگی‌های کلیدی

### 🎨 طراحی و تجربه کاربری
- 🌙 **تم تیره مدرن (Dark Mode)** با پالت رنگی گرم نارنجی-عسلی
- 📱 **کاملاً واکنش‌گرا (Responsive)** — موبایل، تبلت، دسکتاپ
- 🎭 **انیمیشن‌های نرم و حرفه‌ای** برای تعامل بهتر کاربر
- 🅰️ **فونت فارسی Vazirmatn** برای خوانایی بالا
- ↩️ **پشتیبانی کامل از RTL** (راست به چپ)
- 🌈 **گرادینت‌ها و سایه‌های نرم** الهام‌گرفته از طراحی Apple و Linear

### 📊 داشبورد هوشمند
- 📈 نمودار خطی SVG برای تحلیل فروش هفتگی
- 📉 Sparkline در کارت‌های KPI برای نمایش روند سریع
- 💰 شاخص‌های کلیدی: درآمد، سفارش، رزرو، مشتری
- 🔄 سوئیچ بازه زمانی (روز / هفته / ماه)

### 🍕 مدیریت محتوای رستوران
- 📋 مدیریت سفارش‌های لحظه‌ای با وضعیت‌های متعدد
- 🍔 مدیریت کامل منو با دسته‌بندی
- 🪑 سیستم رزرو میز
- 👥 پروفایل مشتریان و تاریخچه
- 👨‍🍳 مدیریت کارکنان و شیفت‌ها
- 📊 پرفروش‌ترین آیتم‌ها

### ⚡ امکانات حرفه‌ای
- 🔍 جستجوی سراسری با میانبر `⌘ + K`
- 🔔 سیستم اعلان‌های زنده
- 🎯 فیلترهای پیشرفته (chip)
- 📑 جدول داده‌های قابل اسکرول
- 🏷️ سیستم وضعیت رنگی (Pending / Cooking / Done / Cancel)

## 🛠️ تکنولوژی‌های استفاده‌شده

| تکنولوژی | کاربرد |
|----------|--------|
| **HTML5** | ساختار سمانتیک |
| **CSS3** | استایل‌دهی، گرید، فلکس، گرادینت، متغیرهای CSS |
| **JavaScript (Vanilla)** | تعامل، رندر داده، نمودار SVG |
| **Vazirmatn Font** | فونت فارسی مدرن از Google Fonts |
| **SVG Inline** | آیکون‌ها و نمودارها |

✅ **بدون هیچ وابستگی خارجی!** — این پروژه به هیچ کتابخانه‌ای مثل React، Vue، jQuery، Bootstrap یا Chart.js نیاز ندارد.

## 🚀 راه‌اندازی

این پروژه **استاتیک** است و به سرور یا نصب چیزی نیاز ندارد:

### روش ۱: باز کردن مستقیم
کافی است فایل `index.html` را در مرورگر باز کنید.

### روش ۲: اجرا با Live Server
```bash
# با استفاده از npx
npx serve .

# یا با پایتون
python3 -m http.server 8000
```
سپس مرورگر را به آدرس `http://localhost:8000` بفرستید.

## 📐 سفارشی‌سازی

تمام رنگ‌ها، اندازه‌ها و افکت‌ها از طریق **متغیرهای CSS** در ابتدای `style.css` قابل تغییر هستند:

```css
:root {
  --primary: #ff6b35;
  --radius: 16px;
  --font: 'Vazirmatn', sans-serif;
  /* ... */
}
```

با تغییر یک متغیر، تمام پروژه به‌روز می‌شود. 🎯
