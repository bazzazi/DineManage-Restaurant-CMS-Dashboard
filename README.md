# 🍽️ سیستم مدیریت رستوران (Restaurant CMS)

یک سیستم مدیریت محتوای سبک، سریع و واکنش‌گرا برای وب‌سایت رستوران که با استفاده از HTML، CSS و JavaScript خالص توسعه داده شده است.

---

## 📌 معرفی پروژه

این پروژه یک پنل مدیریتی ساده برای کنترل و مدیریت محتوای یک وب‌سایت رستورانی است. هدف آن ارائه یک رابط کاربری تمیز، سبک و بدون وابستگی به فریم‌ورک‌های سنگین است تا به راحتی در پروژه‌های کوچک و متوسط قابل استفاده و توسعه باشد.

---

## ✨ ویژگی‌ها

- طراحی کاملاً واکنش‌گرا (Responsive)
- رابط کاربری ساده و کاربرپسند
- بدون استفاده از فریم‌ورک‌های سنگین
- توسعه‌پذیر و قابل شخصی‌سازی
- مناسب برای پروژه‌های آموزشی و واقعی
- عملکرد سریع و سبک

---

## 🛠️ تکنولوژی‌های استفاده شده

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📁 ساختار پروژه

```

cms-build/
│
├── index.html        # صفحه اصلی پروژه
├── css/
│   └── style.css     # فایل استایل‌ها
├── js/
│   └── app.js       # منطق و اسکریپت‌ها
├── assets/           # تصاویر و فایل‌های استاتیک
└── README.md

````

---

## 🚀 نحوه اجرا

1. کلون کردن ریپازیتوری:
```bash
git clone https://github.com/your-username/restaurant-cms.git
````

2. ورود به پوشه پروژه:

```bash
cd restaurant-cms/cms-build
```

3. باز کردن فایل `index.html` در مرورگر

* ذخیره‌سازی داده‌ها در دیتابیس
* داشبورد مدیریتی پیشرفته
