# 🍽️ DineManage – Restaurant CMS Dashboard

یک داشبورد مدیریت محتوای سبک و واکنش‌گرا برای وب‌سایت رستوران

## 📌 معرفی پروژه

**DineManage** یک پنل مدیریتی ساده و سریع برای مدیریت بخش‌های مختلف یک وب‌سایت رستورانی است.  
این پروژه با هدف ارائه یک رابط کاربری تمیز، سبک و بدون وابستگی به فریم‌ورک‌های سنگین طراحی شده است.

---

## ✨ ویژگی‌ها

- طراحی مدرن و واکنش‌گرا (Responsive)
- رابط کاربری ساده و کاربردی
- ساختار سبک و سریع
- بدون استفاده از فریم‌ورک‌های سنگین
- مناسب برای پروژه‌های آموزشی و واقعی
- قابل توسعه و شخصی‌سازی

---

## 🛠️ تکنولوژی‌ها

- HTML5
- CSS3
- JavaScript (Vanilla JS)
