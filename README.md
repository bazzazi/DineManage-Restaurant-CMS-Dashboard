## 📖 معرفی

**DineManage** یک پنل مدیریت محتوای (CMS) اختصاصی و کاملاً شخصی‌سازی‌شده برای صنعت **رستوران، کافه و فست‌فود** است. این پروژه با طراحی مدرن، تیره و حرفه‌ای، الهام‌گرفته از بهترین داشبوردهای جهانی ساخته شده است.

این پنل با استفاده از **HTML, CSS و JavaScript خالص** (بدون هیچ فریمورک یا کتابخانه خارجی) توسعه یافته تا سبک، سریع و قابل سفارشی‌سازی باشد. مدیران رستوران می‌توانند تمام جنبه‌های کسب‌وکار خود را در یک رابط کاربری زیبا و کارآمد مدیریت کنند: از سفارش‌های لحظه‌ای گرفته تا تحلیل فروش، مدیریت منو، رزرو میز و کارکنان.

---

## ✨ ویژگی‌های کلیدی

### 🎨 طراحی و تجربه کاربری
- **تم تیره مدرن (Dark Mode)** با پالت رنگی گرم نارنجی-عسلی
- **کاملاً واکنش‌گرا (Responsive)** — موبایل، تبلت، دسکتاپ
- **انیمیشن‌های نرم و حرفه‌ای** برای تعامل بهتر کاربر
- **فونت فارسی Vazirmatn** برای خوانایی بالا
- **پشتیبانی کامل از RTL** (راست به چپ)
- **گرادینت‌ها و سایه‌های نرم** الهام‌گرفته از طراحی Apple و Linear

### 📊 داشبورد هوشمند
- نمودار خطی SVG برای تحلیل فروش هفتگی
- Sparkline در کارت‌های KPI برای نمایش روند سریع
- شاخص‌های کلیدی: درآمد، سفارش، رزرو، مشتری
- سوئیچ بازه زمانی (روز / هفته / ماه)

### 📋 مدیریت محتوای رستوران
- مدیریت سفارش‌های لحظه‌ای با وضعیت‌های متعدد
- مدیریت کامل منو با دسته‌بندی
- سیستم رزرو میز
- پروفایل مشتریان و تاریخچه
- مدیریت کارکنان و شیفت‌ها
- پرفروش‌ترین آیتم‌ها

### ⚡ امکانات حرفه‌ای
- جستجوی سراسری با میانبر `⌘ + K`
- سیستم اعلان‌های زنده
- فیلترهای پیشرفته (chip)
- جدول داده‌های قابل اسکرول
- سیستم وضعیت رنگی (Pending / Cooking / Done / Cancel)
